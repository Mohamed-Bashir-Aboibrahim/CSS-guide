Selectors

Element Selector
Selects elements based on the element name.
The element selector selects HTML elements based on the element name. You can select all elements of a specific type on a page with this selector.
p {
  color: red;
}

This paragraph will be red.

Class Selector
Selects elements with a specific class attribute.
The class selector selects elements with a specific class attribute. To select elements with a specific class, write a period (.) character, followed by the class name.
.highlight {
  background-color: yellow;
}

This text is highlighted.

ID Selector
Selects a single element with a specific id attribute.
The id selector uses the id attribute of an HTML element to select a specific element. The id of an element should be unique within a page, so the id selector is used to select one unique element.
#header {
  font-size: 2rem;
  color: blue;
}
Attribute Selector
Selects elements based on an attribute or attribute value.
The attribute selector is used to select elements with a specified attribute. You can also specify a value to match against the attribute's value.
/* Selects all elements with a target attribute */
[target] {
  border: 1px solid black;
}

/* Selects all elements with target="_blank" */
[target="_blank"] {
  background-color: yellow;
}

Box Model

width & height
Sets the width and height of an element.
The width and height properties are used to set the width and height of an element. The width and height properties do not include padding, borders, or margins.
div {
  width: 300px;
  height: 200px;
}
margin & padding
Sets the margin and padding space around elements.
The margin property sets the margin space around an element. The padding property sets the padding space inside the border of an element. Both properties can have 1-4 values for different sides.

Shorthand Values:

  • 1 value - Applies to all four sides
  • 2 values - First for top/bottom, second for left/right
  • 3 values - First for top, second for left/right, third for bottom
  • 4 values - Top, right, bottom, left (clockwise)
div {
  margin: 10px 20px; /* top/bottom left/right */
  padding: 15px; /* all sides */
}
border
Sets the border properties for an element.
The border property is a shorthand property for border-width, border-style, and border-color. You can also set these properties individually for more control.
div {
  border: 2px solid #d4a017;
}

/* Individual properties */
div {
  border-width: 2px;
  border-style: dashed;
  border-color: red;
}

box-sizing
Defines how the width and height of an element are calculated.
The box-sizing property defines how the width and height of an element are calculated: should they include padding and borders, or not.

Values:

  • content-box - Default. Width/height only includes content
  • border-box - Width/height includes content, padding and border
div {
  box-sizing: border-box;
  width: 300px;
  padding: 20px;
  border: 5px solid black;
/* Total width remains 300px */
}

Typography

font-family
Specifies the font for an element.
The font-family property specifies the font for an element. The property can hold several font names as a "fallback" system. If the browser does not support the first font, it tries the next font.
body {
  font-family: "Arial", "Helvetica", sans-serif;
}

This text is in Arial font.

This text is in Times New Roman.

font-weight
Sets the weight (boldness) of the font.
The font-weight property sets how thick or thin characters in text should be displayed. It can be specified as a keyword (normal, bold) or a numeric value (100-900).
.bold-text {
  font-weight: bold;
}

.light-text {
  font-weight: 300;
}

Specifies the horizontal alignment of text.
The text-align property specifies the horizontal alignment of text in an element. It can align text to the left, right, center, or justify it.
.center {
  text-align: center;
}

.right {
  text-align: right;

line-height
Sets the height of a line box.
The line-height property specifies the height of a line. It's commonly used to set the distance between lines of text. A unitless value is recommended as it's relative to the current font size.
p {
  line-height: 1.5;

Layout

display
Specifies the display behavior of an element.
The display property specifies the display behavior (the type of rendering box) of an element. In HTML, the default display property value is taken from the HTML specifications or from the browser/user default style sheet.

Common Values:

  • block - Displays as a block element
  • inline - Displays as an inline element
  • inline-block - Displays as inline but with block properties
  • flex - Displays as a block-level flex container
  • grid - Displays as a block-level grid container
  • none - Element is not displayed
.inline-block {
  display: inline-block;
}
position
Specifies the type of positioning method used for an element.
The position property specifies the type of positioning method used for an element (static, relative, absolute, fixed, or sticky).

Values:

  • static - Default. Elements render in order
  • relative - Positioned relative to normal position
  • absolute - Positioned relative to nearest positioned ancestor
  • fixed - Positioned relative to viewport
  • sticky - Toggles between relative and fixed
.fixed-header {
  position: fixed;
  top: 0;
  width: 100%;
}
flexbox
Provides an efficient way to layout, align and distribute space among items.
Flexbox is a layout mode that provides an efficient way to lay out, align and distribute space among items in a container, even when their size is unknown or dynamic.
.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

Creates grid-based layouts with rows and columns.
CSS Grid Layout excels at dividing a page into major regions or defining the relationship in terms of size, position, and layer, between parts of a control built from HTML primitives.

  display: grid;
  grid-template-columns: repeat(3, 1fr);

Backgrounds

background-color
Sets the background color of an element.
The background-color property sets the background color of an element. The background of an element is the total size of the element, including padding and border (but not the margin).
body {
  background-color: #f0f0f0;
}
Element with background color
background-image
Sets one or more background images for an element.
The background-image property sets one or more background images for an element. By default, a background-image is placed at the top-left corner of an element, and repeated both vertically and horizontally.
.hero {
  background-image: url("hero.jpg");
  background-size: cover;
}
background
Shorthand property for all background properties.
The background property is a shorthand property for setting all background style properties at once, such as color, image, origin and size, or repeat method.
body {
  background: #f0f0f0 url("bg.png") no-repeat center center;
}

Transforms & Transitions

transform
Applies 2D or 3D transformations to an element.
The transform property applies a 2D or 3D transformation to an element. This property allows you to rotate, scale, move, skew, etc., elements.

Common Functions:

  • rotate() - Rotates an element
  • scale() - Changes the size of an element
  • translate() - Moves an element
  • skew() - Skews an element
.box {
  transform: rotate(45deg) scale(1.2);
}
transition
Creates smooth transitions between property changes.
The transition property is a shorthand property for transition-property, transition-duration, transition-timing-function, and transition-delay. It allows you to define the transition between two states of an element.
.box {
  transition: all 0.3s ease;
}

.box:hover {
  transform: scale(1.1);
}

Animations

@keyframes
Specifies the animation code.
The @keyframes rule specifies the animation code. The animation is created by gradually changing from one set of CSS styles to another. During the animation, you can change the set of CSS styles many times.
@keyframes example {
  from { background-color: red; }
  to { background-color: yellow; }
}

div {
  animation: example 4s infinite;
}
animation
Shorthand property for animation properties.
The animation property is a shorthand property for animation-name, animation-duration, animation-timing-function, animation-delay, animation-iteration-count, animation-direction, animation-fill-mode, and animation-play-state.
div {
  animation: slidein 3s ease-in-out 1s infinite alternate;
}
